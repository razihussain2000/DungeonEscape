DungeonEscape
DungeonEscape is a top-down, procedural horror-survival game built with Phaser 3. Players must navigate a dark, abandoned castle, carefully managing their limited light source to survive against supernatural threats.


🕹️ Gameplay Overview
Objective: Explore the procedural depths of the castle and find the exit alive.

Core Mechanic: The player's oil lamp serves as both a beacon for safety and a target for enemies. Manage your "light budget" to survive.

Style: Atmospheric, orthographic survival horror.


🛠️ Technical Stack
Engine: Phaser 3 (WebGL Renderer)

Architecture: Entity-Component-System (ECS) inspired modular design.

Assets: Custom texture atlas for modular environmental generation.


🚀 Setup & Development
Clone this repository to your local machine.

Ensure you have a local web server (e.g., Live Server for VS Code) configured to serve the index.html file.

Open the project in your browser via the local server address.
